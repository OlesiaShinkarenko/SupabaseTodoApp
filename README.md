## Supabase TodoApp
Android-приложение для управления задачами через Supabase. Реализованы загрузка списка, добавление, редактирование и удаление задач с авторизацией.
## Возможности
- Загрузка задач пользователя через GET-запрос с фильтром по userId.
- Добавление задач через FloatingActionButton и диалоговое окно (title, priority, due_date).
- Редактирование по долгому нажатию на элемент списка (PATCH-запрос).
- Удаление по кнопке с подтверждением (DELETE-запрос).
- Авторизация через accessToken из SharedPreferences.
